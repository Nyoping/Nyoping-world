이 문서는 서버의 직업 시스템 전반을 정리한 최상위 대표본이다.

## 0. 연동 문서

## 1. 기본 직업: 백수
* 서버에 처음 접속한 모든 플레이어는 기본적으로 백수 상태로 시작한다.

## 2. 직업 공통 등급 시스템

### 2.1 공통 직업 등급
신입 / 폐급 / 초급 / 중급 / 상급 / 최상급 / 특급 / 짬킹

### 2.2 승급 조건

| 등급 | 승급 조건(숙련도 레벨) | 승급 조건(직업 퀘스트) |
|------|----------------------|---------------------|
| 신입 | 0 | 직업별 별도 기재 |
| 폐급 | 5 | 직업별 별도 기재 |
| 초급 | 10 | 직업별 별도 기재 |
| 중급 | 20 | 직업별 별도 기재 |
| 상급 | 30 | 직업별 별도 기재 |
| 최상급 | 50 | 직업별 별도 기재 |
| 특급 | 70 | 직업별 별도 기재 |
| 짬킹 | 100 | 직업별 별도 기재 |

공통 규칙:
* 숙련도 레벨 + 직업 퀘스트 모두 충족해야 승급

## 3. 숙련도 레벨 시스템(공통)

### 3.1 기본 구조
* 모든 직업은 숙련도 레벨 0~100
* 숙련도는 직업별 행동 및 직업 퀘스트를 통해 획득

### 3.2 레벨업 필요 숙련도
모든 구간 100 고정

### 3.3 계산식
필요 숙련도 = 100 (현재 레벨과 무관)

## 4. 공통 아이템 등급 시스템
* 바닐라 / 고급(×1.5) / 최고급(×2.0)
* 고급/최고급은 조합 불가, 요리 재료 투입 불가
* 음식류 섭취 가능, 판매 가능
* 광물류: 제련 가능, 등급 유지
